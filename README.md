# 先进的饰品系统  
**这个Minecraft模组允许你将任何物品（包括所有模组）放入头盔插槽，并包含一些有趣而实用的附加功能。**  
 - 在原版MC中你需要通过指令来获取可放置于盔甲插槽的其他物品，本模组也仅是解除了这一限制（目前只是头盔插槽），至于物品放在头上的呈现效果还需要依赖资源包（没有经过调整的头戴模型可能看起来只是掉落物模型，或者直接内嵌在脑袋里。  
 - 强烈建议与**Cosmetic Armor**模组搭配，这个模组为四个部位增加了一个饰品盔甲位，这样你就可以同时佩戴盔甲和那些乱七八糟的帽子了。（下面的演示均搭配该模组）

## 附加功能
### 隐形盔甲  
 - 如果你不希望盔甲挡住酷酷的皮肤或是你那些奇怪的帽子，只要点击对应盔甲槽旁边的小按钮（只有生存模式物品栏）就能使盔甲隐形了（盔甲属性会保留），按钮为“X”的时候隐藏，为“O”的时候显示。

 - 如果骑在你头上的那个家伙想赖着不走，只需要将鞍脱掉就能强行将TA赶下来
 - 已知问题：
      - 玩家作为乘客时，其碰撞箱尺寸不会改变。所以乘客和载体都是玩家时，载体会因为乘客的碰撞箱过大而挡住视线，导致无法与周围交互（还可能失手打死头上的那个家伙）。
          - 目前的解决方案是使用**ShoulderSurfing（越肩视角）**模组，该模组能够调整摄像机位置。
## 兼容性
 - 前面提到了，它兼容**Cosmetic Armor**，具体版本号是Cosmetic Armor Update-1.21.11-1.7.0
 - 其他饰品模组的兼容性未知
 - 无需前置
